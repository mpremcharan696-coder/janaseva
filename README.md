
SchemeSetu
The Unified Navigator for Government Schemes & Citizen Entitlements.

📌 Problem Statement
Millions in government aid and private welfare go unclaimed every year. Citizens—including farmers, students, small business owners, and senior citizens—miss out on life-changing benefits simply because they don't know they are eligible. The problem is a **Discovery & Navigation Crisis**: the safety net exists, but the path to find it is too complex.

 💡 The Solution
SchemeSetu acts as a "Digital Citizen Compass." By entering a single profile, any citizen can instantly filter through thousands of Central, State, and private welfare schemes to show:
1. Precise Entitlement Matching: A clear list of only the benefits they qualify for.
2. The Step-by-Step Roadmap: A personalized navigation guide to complete the application process for each scheme successfully.

🚀 Key Features
* Universal Citizen Profiler:A one-time form for age, income, occupation, and region details.
* Intelligent Welfare Engine:Uses logic to map user profiles against complex scheme rules (e.g., SSP, PM-Kisan, Pension schemes).
* Guided Application Roadmap:Provides a personalized checklist for each scheme, from document prep to submission.
* Integrated Document Vault:Maps the local documents (Income, Domicile, etc.) needed for specific applications.
* Direct Access & Deep Links:One-click navigation to official application forms, bypassing confusing web portals.
